# ESP32-C3 BLE Digital Potentiometer Controller

Управление цифровыми потенциометрами MCP4011 через BLE на плате ESP32-C3 Super Mini.

## 📋 Возможности

- **BLE UART сервер** с именем "esp32"
- **Управление двумя потенциометрами MCP4011** (0-63 шагов)
- **Эмуляция нажатия кнопки** через оптопару с регулируемой длительностью
- **Светодиодная индикация** состояния подключения
- **Поддержка стандартных BLE терминалов** (nRF Connect, LightBlue, Serial Bluetooth Terminal)

## 🔧 Аппаратное обеспечение

### Компоненты
- ESP32-C3 Super Mini
- 2x MCP4011 (цифровой потенциометр)
- Оптопара (для эмуляции кнопки)
- Светодиод (встроенный на GPIO8)

### Подключение

| ESP32-C3 Pin | MCP4011 #1 | MCP4011 #2 | Оптопара |
|--------------|------------|------------|----------|
| GPIO2 (CS1)  | CS         | -          | -        |
| GPIO3 (CS2)  | -          | CS         | -        |
| GPIO4 (UD)   | U/D        | U/D        | -        |
| GPIO1 (OPTO) | -          | -          | Управление |

## 📱 Как подключиться

1. **Установите BLE терминал** на телефон:
   - Android: [Serial Bluetooth Terminal](https://play.google.com/store/apps/details?id=de.kai_morich.serial_bluetooth_terminal)
   - iOS: [LightBlue](https://apps.apple.com/app/lightblue/id557428110)

2. **Включите Bluetooth** на телефоне

3. **Подключитесь** к устройству `esp32c`

4. **При успешном подключении** получите сообщение:  `Connect - OK`


## 📝 Команды управления

### Формат
`$ <значение1> <значение2> [длительность]`


### Параметры
| Параметр | Диапазон | Описание |
|----------|----------|----------|
| значение1 | 0-63 | Положение первого потенциометра |
| значение2 | 0-63 | Положение второго потенциометра |
| длительность | 50-2000 мс | Длительность нажатия (опционально) |

### Примеры команд

| Что отправить | Что произойдет |
|---------------|----------------|
| `$30 45` | Установка 30 и 45, нажатие 150 мс |
| `$63 63 100` | Максимум, короткое нажатие 100 мс |
| `$0 0 500` | Минимум, нажатие 500 мс |
| `$20 50 1000` | Установка 20 и 50, нажатие 1 сек |

## 💬 Ответы сервера

| Ответ | Значение |
|-------|----------|
| `Connect - OK` | Успешное подключение |
| `++Pots set to: 30 45, click: 150ms` | Команда выполнена |
| `++Error: Use format $value1 value2 [duration]` | Неверный формат |
| `++текст` | Эхо-ответ на обычное сообщение |

## 🔍 Диагностика

### Светодиод (GPIO8)
| Состояние светодиода | Что означает |
|---------------------|--------------|
| Мигает | Нет подключения к BLE |
| Выключен | Есть подключение |


## ⚠️ Возможные проблемы

| Проблема | Решение |
|----------|---------|
| Не вижу устройство `esp32` | Перезагрузите плату (отключите питание на 5 сек) |
| Подключился, но нет ответа | В терминале подпишитесь на нотификации (иконка колокольчика) |
| Команда не выполняется | Проверьте формат: пробелы между числами, знак `$` в начале |

## 🎯 Быстрый старт

1. Загрузите прошивку на ESP32-C3
2. Подключитесь к `esp32c` через BLE терминал
3. Отправьте `$30 45`
4. Потенциометры установятся в положение 30 и 45
5. Произойдет "нажатие" (замыкание оптопары)
