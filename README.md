# ESP32-C3 BLE Digital Potentiometer Controller

Управление цифровыми потенциометрами MCP4011 через BLE на плате ESP32-C3 Super Mini.

## 📋 Возможности

- **BLE UART сервер** с именем "esp32"
- **Управление двумя потенциометрами MCP4011** (0-63 шагов)
- **Эмуляция нажатия кнопки** через оптопару с регулируемой длительностью
- **Светодиодная индикация** состояния подключения
- **Поддержка стандартных BLE терминалов** (nRF Connect, LightBlue, Serial Bluetooth Terminal)

## 🔧 Аппаратное обеспечение

### Компоненты
- ESP32-C3 Super Mini
- 2x MCP4011 (цифровой потенциометр)
- Оптопара (для эмуляции кнопки)
- Светодиод (встроенный на GPIO8)

### Подключение

| ESP32-C3 Pin | MCP4011 #1 | MCP4011 #2 | Оптопара |
|--------------|------------|------------|----------|
| GPIO2 (CS1)  | CS         | -          | -        |
| GPIO3 (CS2)  | -          | CS         | -        |
| GPIO4 (UD)   | U/D        | U/D        | -        |
| GPIO1 (OPTO) | -          | -          | Управление |

### Принципиальная схема
ESP32-C3 Super Mini
┌─────────────────┐
│ │
│ GPIO2 ──────────┼──── CS1 (MCP4011 #1)
│ GPIO3 ──────────┼──── CS2 (MCP4011 #2)
│ GPIO4 ──────────┼──── U/D (оба MCP4011)
│ GPIO1 ──────────┼──── Оптопара
│ GPIO8 ──────────┼──── LED (встроенный)
│ │
└─────────────────┘

## 📦 Установка

### Arduino IDE

1. Установите поддержку ESP32-C3 в Arduino IDE
2. Установите библиотеку **ESP32 BLE Arduino**
3. Скопируйте код из `src/main.cpp`
4. Выберите плату: `ESP32C3 Dev Module`
5. Загрузите скетч
